01

Investigación y Conceptos

02

01

Investigación y Conceptos


Realizo una busqueda de tendencias, referentes, conceptos, etc. Todo aquello que me pueda ayudar para iniciar caa proyexto. Es una forma de renovación y de dar inicio a nuevas ideas. Para esto, según el proyecto, tambien me gusta apoyarme de lluvias de ideas y otras metodologías relevantes.

02

Wireframes


Una vez recopilada toda la información basica del proyecto, y habiendo realizado la investigación visual. Comienzo a realizar maquetas simples de diagramación y posibles modulos del proyecto.

03

Prototipos y/o Guías


Una vez se haya llegado a una definición según el paso anterior, se realizan prototipos o MockUps, User Flow y Task Flow. Para esto se puede usar Ai, Figma, Adobe XD u otra plataforma de UI. Una vez se tienen listos, se van definiendo con el cliente y así poder pasar a la siguiente etapa. Para facilitar un buen entendimiento y alineamiento con el resto del equipo de trabajo. Se pueden realizar las guías pertinentes según el proyecto. Brand Guide, Style Guide, Marketing Guide, UI Design Guide, entre otras.

04

Producción


Teniendo las cosas más claras, se puede realizar el proceso de producción. En donde se crean proyectos solidamente pensados y coordinados con los diferentes miembros del equipo y/o empresa.

05

Testing y Responsive


Para garantizar un diseño adaptable y eficiente. Se testean con los usuarios y en diferentes pantallas (celulares, tablet, computadores de escritorios y pantallas más grandes). Una vez se revisa, un modulo o proyecto en su totalidad, se da por finalizado exitosamente junto a los clientes.
