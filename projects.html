Procesos de Trabajo

  • Diseño Gráfico
  • Diseño Web

* Si un proyecto es extenso, antes de comenzar todo el flujo de trabajo, se llega a un acuerdo con el cliente, respecto a plazos y presentaciones de entregas.

01

Investigación y Conceptos


Se realiza una búsqueda de tendencias, referentes, conceptos, etc. Todo aquello que pueda ayudar para iniciar un nuevo proyecto. Es una forma de renovación y de dar inicio a nuevas ideas. Para esto, según el proyecto, utilizo Brainstorming y otras metodologías relevantes.

02

Wireframes y Bocetos


Una vez recopilada toda la información básica, y, habiendo realizado la investigación visual, se inician las maquetas simples de diagramación y/o posibles módulos del proyecto.

03

Propuestas


Teniendo las cosas más claras, alineados con el cliente, se inicia el proceso de producción. Etapa donde se crean y presentan diferentes propuestas.

04

Finalización


Para finalizar, y con la elección definida, se preparan los documentos en los formatos solicitados para ser entregados satisfactoriamente.

* Si un proyecto es extenso, antes de comenzar todo el flujo de trabajo, se llega a un acuerdo con el cliente, respecto a plazos y presentaciones de entregas.

01

Investigación y Requerimiento


Se realiza una búsqueda de tendencias, referentes, conceptos, etc. Todo aquello que pueda ayudar para iniciar un nuevo proyecto. Es una forma de renovación y de dar inicio a nuevas ideas. Para esto, según el proyecto, utilizo Brainstorming y otras metodologías relevantes con el cliente.

02

Wireframes


Una vez recopilada toda la información básica del proyecto, y habiendo recibido los requerimientos específicos del cliente, se inician las maquetas simples de diagramación y/o posibles módulos del proyecto.

03

Prototipos y/o Guías


Cuando se ha llegado a una definición según el paso anterior, se realizan prototipos o MockUps, User Flow, Task Flow, entre otros (según sea necesario).

Para esto se puede usar Ai, Figma, Adobe XD u otra plataforma de UI. Al terminar las maquetas, se analizan en conjunto con el equipo de trabajo o cliente, para así poder pasar a la siguiente etapa.

Para facilitar un buen entendimiento y alineamiento con el resto del equipo de trabajo, se pueden realizar las guías pertinentes según el proyecto. Brand Guide, Style Guide, Marketing Guide, UI Design Guide, entre otras.

04

Producción


Teniendo las definiciones de los módulos a trabajar, se puede realizar el proceso de producción. En el cual se crean proyectos pensados y coordinados con los diferentes miembros del equipo y/o empresa.

05

Testing y Responsive


Para garantizar un diseño adaptable y eficiente, este se testea con los usuarios y/o clientes en diferentes pantallas (celulares, tablets, computadores y pantallas más grandes). Terminada la revisión del diseño en su totalidad, se da por finalizado exitosamente.
